RILCOT-OS-V0 is a cutting-edge operating system designed for reliability, performance, and scalability.

## Architecture
RILCOT-OS-V0 follows a modular architecture that allows for easy updates and maintenance. The system is built on a microkernel structure, promoting efficiency and scalability.

## Installation
To install RILCOT-OS-V0, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Amdkn/RILCOT-OS-V0.git
   ```
2. Navigate to the project directory:
   ```bash
   cd RILCOT-OS-V0
   ```
3. Run the installation script:
   ```bash
   ./install.sh
   ```

## Configuration
After installation, configure the operating system by editing the configuration files located in `/etc/rilcot/`.

## Deployment with Coolify
To deploy RILCOT-OS-V0 using Coolify, follow these instructions:
1. Ensure you have Coolify installed on your server.
2. Create a new service in Coolify and link it to the repository.
3. Configure environment variables as needed.
4. Click on Deploy.
